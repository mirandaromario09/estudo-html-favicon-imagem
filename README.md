# Estudo HTML - Favicon e Imagem

Exercício de HTML abordando:

- Inserção de **favicon** em páginas HTML
- Exibição de **imagens** no conteúdo da página

## Tecnologias

- HTML5
- Favicon (.ico)
- Imagens (.png)

## Como visualizar

Abra o arquivo `index.html` em qualquer navegador moderno.
